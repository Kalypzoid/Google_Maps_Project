## Frontend Nanodegree Google Maps Project ##

This is a map showing the location of bars in the East Charlotte area. 

You can hover over a marker to see a snippet of a review, and you can click on the marker to open up a list on the side showing a list of bars and a "Yelp it" button that directs youto the Yelp page of the selected bar/location.

You can also use the search bar at the top right to search for another type of business or location if you'd like.

To run this app, just dowload this repository and open up the index file with your desired browser.
